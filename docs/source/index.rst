mvlogics
========

**PyPI package name**: `mvlogics <https://pypi.org/p/mvlogics>`__

A collection of many-valued logics in Python. In classic logical calculi, propositions evaluate to either true or false. In contrast, many-valued
logics offer possibilities in between, the most characteristic example being a value representing "unknown".
